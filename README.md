# Music Player

A simple and responsive Music Player web application that allows users to play, pause, and control audio playback through an intuitive interface. This project demonstrates audio handling, interactive controls, and modern web design using HTML, CSS, and JavaScript.

## Features

* 🎵 Play and pause music
* ⏮️ Previous and next track controls
* 🔊 Volume control
* ⏯️ Audio progress bar
* 📱 Responsive and user-friendly interface
* ⚡ Smooth audio playback

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/music-player.git
   ```

2. Open the project folder.

3. Run the `index.html` file in your browser.
